About TaurusTLS

What is TaurusTLS

TaurusTLS provides OpenSSL 3 and 4 support to the Internet Direct (Indy) component set. TaurusTLS is freely available under an Open Source license (Modified BSD) or MPL (Mozilla Public License) 1.1 allowing you to use it in your commercial projects. TaurusTLS is available for the following platforms:

  • Android
  • iOS
  • Linux
  • OSX
  • Windows including ARM64EC

Both RAD Studio and FreePascal are supported. It includes headers plus components. Those components are:

Component Nme Description
TTaurusTLSIOHandlerSocket Enables TLS in a TIdTCPClientCustom descendant.
TTaurusTLSServerIOHandler Enables TLS in a TIdCustomTCPServer descendant.
TTaurusTLSUnicodeLog Triggers events when data is sent and received and data is provided in Unicode form if supported.

Why TaurusTLS

TaurusTLS is designed to replace Indy's old OpenSSL support. Indy's OpenSSL support is limited to supporting OpenSSL 1.0.2 meaning the following:

  • OpenSSL 1.0.2 reached its end of life in 2019 with no public updates since then.
  • OpenSSL 1.0.2 does NOT support the TLS 1.3 protocol which is the current standard.

In Addition, TaurusTLS offers the following features

  • Virtual hosting where each domain has its own certificate
  • Security Levels where connections are rejected based upon cipher strength or a criterion you use.
