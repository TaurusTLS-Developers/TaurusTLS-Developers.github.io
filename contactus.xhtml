Contacting Us About TaurusTLS

Bug Reports and Pull-Requests

Bug reports and pull requests can be submitted by going to the appropriate repository. There are two repositories. The fist is for the TaurusTLS source-code. The second one contains build scripts for building OpenSSL and publishing the releases. Bug reports about OpenSSL builds are accepted there but please do not report issues with the OpenSSL code itself.

Support Requests

Before asking for support, please make sure you have that latest version. Then consult the README.md file and TaurusForIndy290All.chm HTML Help .CHM file included in the distribution to see if your question can be answered there. Support is provided by the community at the TaurusTLS discussion forums on GitHub. We can not, however offer direct support for Internet Direct (Indy). In addition, you can always ask questions on the Delphi Praxis forums for Indy. Keep in mind that answers come from actual people volunteering their time.

Other Inquiries

Before contacting us, consider if your question can be answered on the appropriate forums and ask there because you may get a quicker answer there. If your inquiry is not appropriate for the forums, you may contact us through E-Mail.
