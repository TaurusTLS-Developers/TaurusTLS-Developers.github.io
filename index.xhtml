About TaurusTLS

What is TaurusTLS

TaurusTLS provides OpenSSL 3 and 4 support to the Internet Direct (Indy) component set. TaurusTLS is freely available under your choice between a Modified BSD or MPL (Mozilla Public License) 1.1 allowing you to use it in your commercial projects. TaurusTLS is available for the following platforms:

  • Android
  • iOS
  • Linux
  • OSX
  • Windows including ARM64EC

TaurusTLS supports the following RAD Studio versions:

  • Delphi 10 Seattle
  • Delphi 10.1 Berlin
  • Delphi 10.2 Tokyo
  • Delphi 10.3 Rio
  • Delphi 10.4 Sydney
  • Delphi 11 Alexandria
  • Delphi 12 Athens
  • Delphi 13 Florence

In addition, FreePascal including the Lazarus IDE are supported.

In addition to header files, TaurusTLS includes the following components:

Components
Component Name Description
TTaurusTLSIOHandlerSocket Enables TLS in a TIdTCPClientCustom descendant.
TTaurusTLSServerIOHandler Enables TLS in a TIdCustomTCPServer descendant.
TTaurusTLSUnicodeLog Triggers events when data is sent and received and data is provided in Unicode form if supported.

Why TaurusTLS

TaurusTLS is designed to replace Indy's old OpenSSL support. Indy's OpenSSL support is limited to supporting OpenSSL 1.0.2 meaning the following:

  • OpenSSL 1.0.2 reached its end of life in 2019 with no public updates since then.
  • OpenSSL 1.0.2 does NOT support the TLS 1.3 protocol which is the current standard.

In Addition, TaurusTLS offers the following features

  • Virtual hosting where each domain has its own certificate
  • Security Levels where connections are rejected based upon cipher strength or a specific criterion.
